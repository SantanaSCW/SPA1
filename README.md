# SPA1 - Portfolio Platform

A React-based Single Page Application (SPA) designed to showcase software engineering projects. This platform allows for adding, searching, and managing portfolio items with persistent storage.

## Features
* **Project Gallery:** Displays all current software projects.
* **Search Functionality:** Real-time filtering by project name.
* **Project Form:** Add new projects with a name and tech stack.
* **Backend Persistence:** Uses JSON Server to save data.

## Setup Instructions
1. **Clone the repository:**
   `git clone <your-repository-url>`
2. **Install dependencies:**
   `npm install`
3. **Start the Backend:**
   `npx json-server db.json --port 3001`
4. **Start the Frontend:**
   `npm start`

## Tech Stack
* React.js
* JSON Server
* CSS (Custom Styling)
